Section 5: 5 More Markups & What are they for

An article element in HTML represents a self-contained piece of content that could be independently distributed or reused. It often contains a single topic or theme and is typically used for things like blog posts, news stories, forum posts, or other independent content pieces.
The div element in HTML is a generic container used to group other HTML elements together. It doesnt have any semantic meaning on its own, but it's commonly used for styling and layout purposes, often in conjunction with CSS and JavaScript.
The span element in HTML is an inline container used to group a portion of text or inline elements for styling or scripting purposes.
The footer element in HTML is used to define the footer section of a document or a section within it.
